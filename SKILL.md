## Quick-reference

- **Fonts**: `Ma Shan Zheng` (brush display, titles, food names, 宜/忌 seals) + `LXGW WenKai TC` (body, all prose). Load both from Google Fonts.
- **Surfaces**:
  - Daylight parchment: `--bg #f5f0e8` with ink `#2c2416`. Use for home, history.
  - Tarot dark: `#1a1612` with `#c4a44e` gold border. Use for result card, devil card.
- **Tier palette** (gold-warm → ascetic → legend):
  - 庶民 `#c4a44e` · 小資 `#d4b44e` · 放縱 `#e8943a` · 苦行 `#7a8088` · 傳說 `#ffd700` + vermillion `#c23a2e`
- **Seal stamps** (bordered rotated calligraphy): 宜, 忌, 御膳, 週五無禁忌, 週三症候群
- **Iconography**: emoji only — 🍚 飽德大師, 🔢 卡路里仙人, 🛵 外送天尊, 🍱 便當真人, 👑 週五大帝, 👹 飢餓魔王
- **Voice**: Traditional Chinese. Deadpan, slightly cruel, never cheerful. Masters are wise AND judgmental. Never use exclamation marks unless cursed.
